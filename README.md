# Real Estate Price Prediction using Linear Regression (Python)

## 📌 Project Overview

This project builds a machine learning model using **Linear Regression** to predict real estate prices based on property features and location data.

The goal is to analyze how different factors influence house prices and to develop a predictive model that estimates the **price per unit area**.

---

## 📊 Dataset Description

The dataset includes the following features:

* **Transaction Date** – Date of property sale
* **House Age** – Age of the property
* **Distance to Nearest MRT Station** – Accessibility to public transport
* **Longitude** – Geographic coordinate
* **Latitude** – Geographic coordinate
* **Number of Convenience Stores** – Nearby facilities
* **House Price per Unit Area** – Target variable

---

## 🎯 Objectives

* Build a Linear Regression model for price prediction
* Analyze the impact of location and accessibility on prices
* Identify key features affecting real estate value
* Evaluate model performance using regression metrics

## 🔍 Project Workflow

### 1. Data Preprocessing

* Handling missing values
* Converting transaction date
* Cleaning and structuring data

### 2. Exploratory Data Analysis (EDA)

* Feature distribution analysis
* Correlation heatmap
* Relationship between variables and price

### 3. Feature Engineering

* Extracting insights from transaction date
* Scaling features if necessary

### 4. Model Building

* Applying **Linear Regression**
* Splitting dataset into training and testing sets

### 5. Model Evaluation

* Mean Squared Error (MSE)
* R² Score

### 6. Visualization

* Actual vs predicted price comparison
* Residual analysis plots

---

## 📈 Key Insights

* Properties closer to MRT stations tend to have higher prices
* More nearby convenience stores increase property value
* Geographic location significantly impacts pricing
* Older houses generally show lower price trends
